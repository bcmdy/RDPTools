## 工具简介

`1Remote` 是一个现代化的个人远程会话管理器和启动器。它把多种远程连接集中到一个地方，便于维护服务器列表、凭据、标签和连接配置。

## 主要功能

- 支持 RDP、SSH、VNC、Telnet、SFTP/FTP、RemoteApp、NoMachine 以及外部应用启动。
- 提供快速启动器，便于快速查找并连接远程会话。
- 支持多屏和 HiDPI RDP 连接。
- 可为连接配置标签、图标、颜色、备注、凭据和脚本等信息。
- 支持导入/导出连接数据，适合在多台机器之间迁移配置。
- 支持以标签页或独立窗口方式管理多个远程会话。

## 使用建议

1. 运行 `1Remote.exe`。
2. 在程序内新增远程连接，按实际协议填写主机、端口、账号、凭据和显示设置。
3. 对常用连接设置标签和备注，后续可通过搜索或快速启动器进入。
4. 迁移配置前，优先使用程序内置的导入/导出功能。

## 注意事项

- 远程连接凭据属于敏感信息，请妥善保护配置文件和备份文件。
